## 三、高赞内容特征分析

### 标题公式
```
[痛点词] + [数字] + [解决方案]
例：「眼睛痒到想抠出来？这3招救我命」

[人群标签] + [场景] + [情绪词]
例：「过敏星人的春天是刑期」

[产品] + [测评/对比] + [结果]
例：「5款抗组胺药真实测评，这款最管用」
```

### 封面特征
- 前后对比图（用药前 vs 用药后）
- 产品平铺图（多款口罩/药物摆拍）
- 症状特写（红眼睛/流鼻涕真实照片）
- 文字大字报（痛点词放大）

### 发布时间规律
- 高发期：3-5月（春季花粉）、8-10月（秋季蒿草）
- 高峰时段：早上 7-9点（起床症状明显）、晚上 9-11点（睡前刷手机）
